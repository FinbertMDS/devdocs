    Test run¶

    • Run all tests 
      cd <magento2_root_dir>/dev/tests/functional
vendor/bin/phpunit
    • Run particular test 
      cd <magento2_root_dir>/dev/tests/functional
vendor/bin/phpunit --filter <name of test>
